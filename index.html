How to use Artificer

Built for the Table

Artificer is designed to handle the heavy lifting, whether you're in the middle of a session or prepping for the next one.

Scenario A

Prep a Session

Load world data into the Atlas, seed locations with quest hooks, and pre-load soundscapes for upcoming scenes. Campaign memory ensures continuity across sessions.

Scenario B

Run Travel

Manage travel states, weather shifts, and regional markers in real-time. Use the Atmosphere Engine to shift the mood as players cross borders.

Scenario C

Resolve an Encounter

Drop into the Tactical Table for grid-based combat. Roll digital dice with modifiers and track the initiative without losing your place in the narrative.

Common Questions

Frequently Asked

Is this tied to one rules system?

No. Artificer is designed as a system-neutral toolkit. While it handles complex logistics, its core modules (Atlas, Journal, Sound) work with any tabletop setting.

Does it replace the GM?

Never. The goal is to remove bookkeeping friction. Artificer supports the GM with structured tools and AI assistance so they can focus on storytelling and player choices.

Is there a playable demo?

We are currently polishing the prototype into a public demo mode. Backers help us reach this milestone faster with sample campaigns and no-login-required access.

What are backers funding?

Backers fund the transition from a internal tool to a public-ready toolkit: UI polish, demo content, AI narrative integration, and the creator-facing world building tools.

Roadmap for backers

A focused release promise.

  1. Public demo path Sample party, sample campaign and no private keys required.
  2. AI GM bridge Tool calls for narration, travel, journal updates and tactical actions.
  3. System-neutral packs Content and wording that keep Artificer useful beyond a single rules brand.
  4. Creator release World, NPC, item and encounter tools polished for other GMs.
Independent Tabletop Toolkit

Artificer is not affiliated with, endorsed by, or partnered with any specific tabletop RPG brand. It is an independent toolkit designed for use with any system.
